✨Available course options✨

“Let the Games Begin”

  (Beginners’ Course)

  • Start with learning Zhuyin (bopomofo) or Pinyin
  • Learn everyday words, phrases and sentences
  • Learn fundamental grammar rules
  • Learn to recognize Chinese characters  
  • Use games to help you remember what you’ve learned
  • Homework and tests to help you practice and improve
  • Complete TOCFL Novice Level (HSK 1/2)

“Shoot the Breeze”

        (Conversation Practice)

  • Practice speaking fluidly about common topics  
  • Learn how to express yourself like a native
  • Learn fun facts along the way
  • No textbook

"他是很高 ❌“

       (Grammar Course)

  • Learn important grammar structures from popular/classic songs
  • Learn how to construct sentences in a natural way
  • Homework and tests to help you practice and improve

Please contact me about additional courses. Before you begin a course, I will assess your current level so that you can start learning appropriate material.
